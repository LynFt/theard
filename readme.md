#多线程和多进程
-   程序：代码以文本形式存入文档

-   进程 ：程序运行的一个状态
        -  包含地址，空间，和数据栈
        -   每个进程完全独立的运行环境，多个进程共享数据是一个问题
  
-   线程：一个进程的独立运行片段，一个进程有多个线程
        -   轻量化的进程
        -   一个进程多个线程共享数据和运行环境
        -   共享互斥问题
 
 -  全局解释器锁
        -   python代码执行是有python虚拟机进行控制
        -   主循环中只有一个控制线程在执行
